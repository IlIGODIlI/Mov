## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please do **NOT** open a public issue. We request that you report it privately to the maintainers at security@example.com instead. 

We will acknowledge your report within 48 hours and work with you to analyze and patch the issue prior to public disclosure.
